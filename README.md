## To build only:

* $ cd build
* $ ./build.sh {sender address}

## To build and install:

* First run FlextesaPlayground ( https://github.com/dani-jozsef/FlextesaPlayground )
* $ cd build
* $ ./install.sh {tezos-client sender account name}
